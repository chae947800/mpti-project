[작성자: 황채리] - 회원가입 및 로그인
역할: Streamlit 웹 개발자 및 파이썬 백엔드 개발자
목표: `pages/auth.py` 파일에 들어갈 로그인 및 회원가입 기능 코드를 작성해줘.

[기본 환경 및 데이터 저장]
1. 가입한 회원들의 정보(이름, 아이디, 비밀번호)는 로컬 폴더에 `users.json` 파일로 저장하고 불러오는 방식으로 구현해줘. (데이터베이스 대신 사용할 목적이며, 앱이 꺼졌다가 켜져도 데이터가 유지되어야 함)
2. `users.json` 파일이 없을 경우를 대비해, 파일이 없으면 자동으로 빈 딕셔너리나 리스트 형태로 생성하는 초기화 함수를 포함해줘.

[UI 구성]
1. Streamlit의 `st.radio` 또는 세련된 화면 전환 방식을 사용해 상단에서 '로그인'과 '회원가입' 모드를 선택할 수 있게 해줘.
2. [로그인 화면]: 아이디, 비밀번호 입력 창과 '로그인' 버튼을 배치해줘. (비밀번호는 type="password" 적용)
3. [회원가입 화면]: 이름, 아이디, 비밀번호 입력 창과 '회원가입' 버튼을 배치해줘.

[기능 및 예외 처리 로직]
1. 회원가입 예외 처리:
   - 사용자가 회원가입 버튼을 눌렀을 때, 입력한 아이디가 이미 `users.json`에 존재한다면 "이미 존재하는 아이디는 이용할 수 없습니다."라는 에러 메시지(st.error)를 보여주고 가입을 막아줘.
   - 중복이 없다면 `users.json`에 이름, 아이디, 비밀번호를 저장하고 "회원가입이 완료되었습니다! 로그인해 주세요."라는 성공 메시지(st.success)를 띄워줘.

2. 로그인 예외 처리:
   - 사용자가 로그인 버튼을 눌렀을 때, 입력한 아이디가 `users.json`에 없다면 "회원가입 먼저 진행해 주세요."라는 메시지를 띄워줘.
   - 아이디는 존재하지만 비밀번호가 틀렸다면 "비밀번호가 일치하지 않습니다."를 띄워줘.
   - 로그인 성공 시, `st.session_state`에 로그인 성공 상태(예: st.session_state['logged_in'] = True)와 유저 정보를 저장하고 환영 메시지를 띄워줘.


[작성자 : 황채리] - 검사 결과 데이터
너는 음악 성격 유형 검사(MPTI) 웹사이트를 만드는 파이썬(Python) 개발자야. 
내가 주는 기획 데이터를 바탕으로, 결과 페이지에서 사용할 데이터를 처리하는 `result_logic.py` 파일의 딕셔너리 코드를 완벽하게 작성해줘.

[구현 조건]
1. 언어: Python
2. 데이터 구조: 각 타입(예: 'HVFN', 'HVFO' 등)을 Key로 하고, 내부 데이터를 Value로 갖는 `mpti_results` 변수를 생성해줘.
3. 중요 조건: 한 유형당 3가지의 서로 다른 성격 결과가 존재해. 따라서 내부 데이터는 3개의 독립된 결과 세트를 가진 '리스트(List)' 구조여야 해. 이 중 하나를 랜덤으로 뽑아 쓸 예정이야.
4. 결과 세트(딕셔너리) 구성 요소:
   - "title": 해당 결과의 캐릭터 이름
   - "stats": 해당 캐릭터의 스탯 항목과 수치 (문자열 리스트 형태)
   - "description": 해당 캐릭터의 성격 유추 문장 (문자열)
   - "image_prompt": 이미지 생성 AI에 사용할 영어 프롬프트 (3D 파스텔 톤, 흰색 배경 스타일로 통일)

[MPTI 알파벳 성격/취향 매칭 기준]
1. H (Heavy: 묵직함, 자극적) vs L (Light: 가볍고 청량함, 편안함)
   - H: 강한 자극, 도파민 추구, 화끈하고 주관이 뚜렷함
   - L: 미니멀리즘, 자극 완화, 평화주의자, 무던하고 편안한 성격
2. V (Vocal: 목소리 중심) vs I (Instrumental: 악기/사운드 중심)
   - V: 인간 중심, 감정 공감, 소통 중시, 타인의 이야기에 귀를 잘 기울임
   - I: 구조 중심, 이성적 분석, 몰입과 집중, 혼자만의 사색과 공간을 중시
3. F (Fast: 빠른 템포) vs S (Slow: 느린 템포)
   - F: 높은 텐션, 즉흥적, 행동파, 지루한 걸 못 참고 성격이 급함
   - S: 신중함, 계획적, 감성적, 여유롭고 깊이 있는 호흡을 선호
4. N (New: 트렌디, 최신) vs O (Old: 레트로, 클래식)
   - N: 트렌드세터, 얼리어답터, 변화 수용, 과거보다 미래와 현재를 봄
   - O: 아날로그 감성, 낭만주의자, 추억 소중히 여김, 익숙함에서 안정감을 느낌
